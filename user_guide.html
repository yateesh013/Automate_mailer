ğŸ›  How to Use

1. Login

Enter your credentials (default: admin / password) to access the dashboard.

2. Setup SMTP

Go to SMTP Settings and enter your Gmail and App Password. This is required for sending emails.

Note: You must enable 2-Step Verification in Gmail and generate an App Password.

3. Group Send with Constant Data

Upload an Excel file containing at least an email column. The same subject and message will be sent to all addresses.

4. Group Send with Dynamic Data

Upload an Excel file with columns like name, email, etc. In your message, use placeholders like: Hello {name}, your ID is {student_id}. The tool will replace them with real values for each recipient.

5. Group Send with API

Upload Excel with student IDs and emails. The tool will fetch extra details (like attendance) from an external API, and then send customized messages only to relevant users.

6. Logs & Refresh

Monitor sending progress in the log panel. After completion, click Refresh to clear logs for a new session.
